程序描述：
一个糖果网上商城。

实现界面
1. default.aspx 主页面
包含内容：
1）导航栏（商标、主页、购买界面、contact、about）
2）推荐商品图片（滑动）
3）网站介绍

2. CandyList.aspx 糖果列表界面
包含内容：
1）根据糖果的数据库将所有糖果显示出来。
2）每个糖果的名字是超链接，点击可进入详情（够买界面）
3）页面右侧有一个菜单栏，显示登陆状态。可通过菜单栏登陆、注册、登出、查看购物车。部分功能只有在登陆后才显示。

3. CandyInfo.aspx糖果详情、购买界面
包含内容：
1）所选糖果的完整信息
2）显示登陆状态的菜单栏
3）购买、加入购物车按钮。点击加入购物车，若成功则会提示当前购物车已有多少该商品。

4. Cart.aspx 购物车界面
包含内容：
购物车中已有的所有商品。

5. Signin.aspx登陆界面
登陆账号，登陆完成后跳转回原界面。
有注册按钮。

6. SignupPage.aspx注册界面
注册界面。注册完成后跳转回登陆界面。

实现功能：
1. 用户注册、登陆功能。登陆状态的保持与更新。用户账户信息的存储与更新。
2. 糖果信息展示功能。
3. 糖果加入购物车功能。
4. 查看当前用户购物车功能。
